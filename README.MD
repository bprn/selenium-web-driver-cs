# README

## Connection String
Sebelumnya ubah terlebih dahulu connection string di masing masing project ke database lokal. Untuk connection string bisa dilihat di file appsettings.json dan app.config.

## Database
Database yang digunakan adalah SQL Server. Untuk membuat database bisa menggunakan script yang ada di folder "Database" pada repository ini. Script tersebut berisi perintah untuk membuat database, tabel, dan data awal yang diperlukan.
